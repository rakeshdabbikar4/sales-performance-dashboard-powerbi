# Sales Performance Dashboard (Power BI) 
## Business Problem
Analyze sales data to identify key trends, top-performing products, and regional performance to support better business decision-making.

## Project Overview
This project presents an interactive Sales Performance Dashboard developed using Microsoft Power BI.  
The dashboard enables analysis of sales performance through key metrics, trends, and product-level insights.

---

## Objectives
- Monitor total revenue, orders, and profit
- Analyze monthly revenue trends
- Identify top revenue-generating products
- Compare regional sales performance
- Enable interactive filtering using slicers

---

## Dashboard Features

### KPIs
- Total Revenue
- Total Orders
- Profit

### Visualizations
- Line Chart: Total Revenue by Order Month
- Bar Chart: Total Revenue by Product Name

### Filters (Slicers)
- Order Date (Date Range)
- State (Region)

### Business Insights
- Best Performing Month: January 2024
- Top Revenue-Generating Product: Laptop
- Lowest Sales Region: Gujarat
- Overall Trend: Consistent upward revenue growth
